# ayx-m-guide

## 项目简介

`ayx-m-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为通用 HTML 页面的存放与索引。

## 目录结构

```
ayx-m-guide/
├── index.html          # 仓库主页入口
├── pages/              # 独立 HTML 页面存放目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立页面，每个页面可包含自己的样式与脚本。
- `assets/` 目录用于存放页面共享或引用的静态资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部服务或特定域名。
- 页面内容保持简洁，避免包含指向不明或未经验证的外部链接。
- 归档页面仅用于信息展示或功能演示，不涉及任何商业或营销用途。

## 维护说明

- 欢迎提交 pull request 添加新的独立 HTML 页面。
- 贡献前请确保页面内容合法、无害，且不包含任何具体域名或推广信息。
- 仓库维护者保留审核与合并的权利。
